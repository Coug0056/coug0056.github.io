Tags I have Learned

Below is a list of some new tags I have learned while taking Responsive Web Design 1 this semester.

The ul Tag
This is a tag that represents an unordered list. You can use it to create lists where the order of the items doesn't matter. However you must tag all the items in the list as li tags when using the ul tag.
The li Tag
This tag is used within the ulul. This is for listing items within the list, and is used after the ul or ol tags, which are the start of ordered or unordered lists. You cannot use these on their own.
The h1 Tag
This tag is used for creating a header on your page. The header with the smallest number will be the most prominent on your page. You can use the tag with other numbers but they will be smaller on the page, and considered less important information.
The head Tag
This is the second tag that you input into your code. This contains important information about your webpage for the computer to read; including the tab name that will be displayed on the webpage.
The img src="your image here" Tag
This is an important tag to include in your code if you want to include an image on your page. This tag tells the computer where your images is and where to insert it on the page. This only works if you have the image in a folder in the same initial file you started working on. This also works with the alt= tag which is an alternative featuring the description of the image. This is if the image cannot show up or the user is using a image description service on your website.
